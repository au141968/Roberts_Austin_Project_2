# Project 2: Responsive Coded Interface
## Project Description
 What did you make?

 #### I made a landing page for a fictitious hot sauce company called Aztec Axolotl.

## Intended User or Player
Who is this interface designed for?

#### It is intended for consumers that want to experience new and interesting hot sauce flavors.

## User / Player Goal
What should the user or player be able to do or understand?

#### The user should be able to see the images of the products, a brief description of each, and a link to further information including price.

## Inspiration Interfaces
List the inspiration interfaces you used and briefly explain what you learned from them.

#### I got my inspiration from two hot sauce websites, including jerseygirlhotsauce.com, and Guy Fieri's Flavortown website. The brightly colored depictions of the bottles with bold lettering, inspired my typography choice. Guy Fieri's website color palette inspired me to use bright, tropical tones. In Jersey Girl's website, the size of the bottle on the webpage, gave them more visual hierarchy.

## Design Choices
Layout:
#### The products logo is at the top, I strategically placed a rainforest image as the main picture to transport the user into that place directly underneath are the pictures of the three products.

Color:
#### I used a tropical color palette, which included the colors used in my logo. These colors included: Pink, turquoise, teal, and red for the typography.

Typography:
#### The typography font was called Aztec Regular. A free download from Resource Boy.

Visual hierarchy:
#### Logo at the top center, large picture of a rainforest to transport the user into that frame of mind. Then, the three products are displayed equally in size.

Feedback states:
#### The hovering buttons below each product's description change from aqua to yellow when you hover over them.

Responsive design:
#### The images will stack vertically when viewed on a smaller screen.
